Once configured, the Vapi voice assistant can be accessed directly from the Odoo frontend via a floating button or systray icon.

This module only makes the assistant from [Vapi.ai](https://www.vapi.ai) available within Odoo’s UI. It allows users to trigger voice interactions, but **does not implement the complete backend logic to interpret and execute complex actions inside Odoo**.

Typical use cases of this base module:
- Open the voice assistant interface from any screen
- Start voice interactions with a pre-configured Vapi assistant
- Basic client-side integration
