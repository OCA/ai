This module is part of the broader effort to enable voice-based interaction with Odoo ERP.

Instead of implementing a custom voice engine inside Odoo, this integration relies on external tools like Vapi.ai, which specialize in real-time voice AI and allow faster evolution and better performance.

This module assumes that speech-to-text and intent resolution are handled externally, and focuses on executing those intents through Odoo’s API or business logic.
