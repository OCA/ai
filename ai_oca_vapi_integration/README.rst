This module integrates Odoo with the Vapi.ai voice assistant platform,
allowing users to interact with Odoo (e.g., CRM, Contacts, Calendar) via
spoken commands.

It injects the Vapi SDK into the frontend and provides the necessary
hooks and endpoints to process conversational actions.

Use Cases / Context
===================

This module is part of the broader effort to enable voice-based
interaction with Odoo ERP.

Instead of implementing a custom voice engine inside Odoo, this
integration relies on external tools like Vapi.ai, which specialize in
real-time voice AI and allow faster evolution and better performance.

This module assumes that speech-to-text and intent resolution are
handled externally, and focuses on executing those intents through
Odoo’s API or business logic.

Configuration
=============

1. Go to **Settings > Technical > Vapi Configuration**.
2. Define your Vapi API key and assistant ID.
3. Enable the voice assistant button from the systray or via frontend
   injection.
4. Optionally, adjust the domain or user permissions if using
   multi-company setup.

Ensure your base URL is accessible externally if used in production,
especially if the assistant needs to make callback requests to your Odoo
instance.

Usage
=====

Once configured, the Vapi voice assistant can be accessed directly from
the Odoo frontend via a floating button or systray icon.

This module only makes the assistant from
`Vapi.ai <https://www.vapi.ai>`__ available within Odoo’s UI. It allows
users to trigger voice interactions, but **does not implement the
complete backend logic to interpret and execute complex actions inside
Odoo**.

Typical use cases of this base module:

- Open the voice assistant interface from any screen
- Start voice interactions with a pre-configured Vapi assistant
- Basic client-side integration

⚠️ If you're looking for a complete Odoo integration where the agent can
**create leads, register meetings, update inventory, manage sales or
purchases** via voice, contact us at `Polpo <https://www.polpo.uy>`__.
We offer a fully working backend integration with customizable logic for
CRM, Sales, Purchases, Inventory, Manufacturing and more.

Known issues / Roadmap
======================

- Add toggle settings to show/hide assistant per user or company
- Improve SDK initialization to support multiple assistants
- Include optional voice trigger to activate assistant (e.g., hotword)
- Add developer hooks to customize assistant initialization logic
- Improve support for multi-company environments

Note: This module only makes the Vapi voice assistant available in the
Odoo interface. It does **not include the logic to handle intent
resolution or execute Odoo operations**. That functionality is handled
externally via API.
